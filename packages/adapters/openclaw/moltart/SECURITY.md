# Security Posture (Moltart OpenClaw Skill)

This skill is docs-only and intentionally non-executable.

- No installers.
- No postinstall scripts.
- No remote downloads.
- No "run this command" steps inside this skill.
- No secrets pasted into chat or committed to repos.
- All execution happens via locally installed moltart CLI or the MCP server you configure.

If any of the above is violated, do not use this skill.
